# Play
Extract the whole game folder. Then just run the main.exe file inside the src folder and you are good to go!

# Build from source
1. **Installing raylib**
   Install [raylib](https://www.raylib.com/) by using this link, the Download Now button guides you to the raylib setup. That installs a lot of good tools to use in raylib development, 
   this is a easy way (and how i basically did the first time i found it) to install raylib.
     
   **OPTIONAL:** You can also install the compiler ([w64devkit](https://github.com/skeeto/w64devkit/releases)) and then install [raylib](https://github.com/raysan5/raylib/releases) all by yourself. For more details follow [this tutorial](https://github.com/raysan5/raylib/wiki/Working-on-Windows).
2.   **Building in Notepad++**
   Open Notepad++ (comes with the raylib installer itself) then open main.c with it. After that, press F6 (or go to Plugins->NppExec->Execute Npp Script), make sure you're using the script "raylib_compile_execute", if you are, then press OK.
